Área

Noción fundamental. Área de un cuadrado de lado 1.

Esta noción fundamental se utiliza para encontrar el área de un rectángulo de lados a,b∈Na,b\in\mathbb{N}a,b∈N: esta área es a⋅ba\cdot ba⋅b.

Esto se generaliza a cualquier rectángulo de lados a,ba,ba,b: el área es a⋅ba\cdot ba⋅b.

Área de un paralelógramo

Problema. Como se vio en el problema 9 del listado 3, un paralelógramo puede ser aumentado a un rectángulo usando dos triángulos rectángulos.

Si los lados de un paralelógramo miden a,ba, ba,b, el ángulo entre ellos mide α\alphaα, y la alturas desde cada lado miden ha,hbh_a, h_bha​,hb​ respectivamente, el área del paralelógramo puede expresarse de las formas siguientes:

  • a⋅haa\cdot h_aa⋅ha​
  • b⋅hbb\cdot h_bb⋅hb​

Relación entre lados, ángulos y áreas

Si se fija una base bbb y una altura hbh_bhb​, hay muchos (infinitos!) paralelógramos distintos que se pueden construir con esos parámetros. Se puede variar la longitud aaa del lado distinto de la base o el ángulo interno γ\gammaγ que forma ese lado con la base. Las restricciones son:

  • a≥hba\ge h_ba≥hb​
  • 0∘<γ<180∘0^\circ<\gamma <180^\circ0∘<γ<180∘

Es importante destacar que aunque ambos parámetros no son constantes, no son independientes el uno del otro: al fijar un valor del ángulo se fija la longitud del lado; y al fijar la longitud del lado el ángulo sólo puede tomar dos valores suplementarios posibles. Notar también que la elección de bbb no influye en este proceso.

Se obtiene la siguiente expresión para el área del paralelógramo:

  • a⋅b⋅sen⁡(γ)a\cdot b\cdot \operatorname{sen}\left(\gamma\right)a⋅b⋅sen(γ)

Área de un triángulo

Idea clave. Dos copias de un mismo triángulo pueden ser combinadas para construir un paralelógramo con la misma base y altura.

Luego el área de un triángulo se puede obtener con expresiones de la forma

  • 12absen⁡γ\frac{1}{2}ab\operatorname{sen}\gamma21​absenγ
